# Numbat

Architectural productivity tools for Rhino 8.

## Installation

1. Open Rhino Package Manager
2. Enable "Include pre-releases"
3. Search for Numbat
4. Install

## Commands

### nbRandomColour
Randomises the colours of file layers.

### nbGFA
Calculates GFA from selected closed polysurfaces by measuring the area of the bottom face of each volume.
